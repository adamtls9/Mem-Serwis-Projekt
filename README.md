# Mem-Serwis-Projekt

## General Information:

Projekt zaliczeniowy na kierunku ALK Frontend Developer z React.
Aplikacja wyświetla listę memów które użytkownicy mogą oceniać.
Memy z najlepszym stosunkiem like do dislike trafiają na listę Top.

## Setup:

git clone
npm install
npm start

## Technologies Used:

React
React Context
CSS Modules
React router

- wersja 1.0 desktop
